showDescription :: Monad m => GameStateM m String Source #

Provide the whole description of the Location the player is in, taking Conditions into consideration if necessary. if Condition for description of the current Location is not met it will not be added to the result.

checkCondition :: Monad m => CondKey -> GameStateM m Bool Source #

Check whetever the given Condition, indexed by CondType and CondId, is true in current state of a game.

checkConditionWith :: Monad m => CondKey -> a -> a -> GameStateM m a Source #

The same as with checkCondition but you can provide what values should be returned on result

checkConditionWith (cl, id) trueValue falseValue

checkConditionWithM :: Monad m => CondKey -> GameStateM m a -> GameStateM m a -> GameStateM m a Source #

The same as with checkConditionWith but in Monad version.

checkConditionWithM (cl, id) (Just truValue) Nothing
